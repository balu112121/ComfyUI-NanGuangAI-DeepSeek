# 南光AI小助理DK – ComfyUI 插件

> 在 ComfyUI 中调用阿里云百炼的 DeepSeek 模型，实现 AI 对话生成。

---

## ✨ 功能特点

- 支持 **DeepSeek-V4-Pro** 和 **DeepSeek-V4-Flash** 模型选择
- 内置 **专家模式** 系统提示，可自定义引导模型思考
- 全中文操作界面，所有输入输出标签均为中文
- 可灵活配置 API Key（输入框或环境变量）
- 错误信息友好，便于调试

---

## 📥 安装步骤

### 1. 下载插件
将 `ComfyUI-NanGuangAI-DeepSeek` 文件夹放入 ComfyUI 的 `custom_nodes` 目录下。
